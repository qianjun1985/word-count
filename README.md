## 英文单词处理工具 v1.0
=====================

使用方法：
1. 双击运行"word_count_gui.exe"或者python环境下双击“word_count_gui.py”
2. 选择输入文件（PDF 或 TXT）
3. 选择输出目录
4. 点击"开始处理"
<img width="1072" height="788" alt="qianjun" src="https://github.com/user-attachments/assets/1df4a012-2613-4eac-8837-05647d83895d" />

依赖说明：
- 如果只在 Python 环境运行.py 文件，想要针对 PDF 操作，需要安装 pip install pdfplumber
- 首次运行可能需要安装 Visual C++ 运行库
- 如被杀毒软件拦截，请添加信任
